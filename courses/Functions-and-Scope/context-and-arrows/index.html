Context and Arrows

⌛ Average time to completion = ~1 hour ⌛

Our introduction to the world of JavaScript functions. We all use functions every day, they are the backbone of our Applications. Find out about context, `this` keyword and how can regular functions become _methods_ in JavaScript classes. Once we understand these topics we will see how ES6 arrow functions bend the rules.
